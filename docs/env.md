# 环境变量

## 数据库

```env
# 数据库地址
mongo_host=
# 数据库端口
mongo_port=
# 数据库账户名称
mongo_username=
# 数据库账户密码
mongo_password=
# 数据库名称
mongo_database=
```

## Redis

```env
# Redis地址
redis_host=
# Redis端口
redis_port=
# Redis密码
redis_password=
# Redis数据库
redis_db=
# Redis过期时间
redis_ttl=
```
